Normal Flow

p태그도 대표적인 block 요소입니다.

div는 의미없이 구역을 나누기 위해서 쓰는 block 태그

block요소는 기본적으로 부모 요소 너비의 100%를 차지하며, 자식 콘텐츠의 최대 높이를 취한다.

block 요소의 총 너비와 총 높이는 content + padding + border(상하/좌우 두께)

block 요소는 서로 margins로 구분된다.(적용하지 않아도 기본적으로 margin을 가진다)

inline 요소는 div태그 처럼 의미없이 구역만 나누는 용도, 자기 콘텐츠의 너비와 높이만 차지한다. 그리고 inline 요소는 width 나 height 속성을 지정할 수 없다.

이미지 요소 또한 inline 요소 중 하나이다. 단, 이미지는 다른 inline 요소들과 달리 width 나 height 값을 지정할 수 있다.

만약 inline 요소의 크기를 제어하려면 block으로 변경해버리거나 inline-block 요소로 설정해주어야 한다.
